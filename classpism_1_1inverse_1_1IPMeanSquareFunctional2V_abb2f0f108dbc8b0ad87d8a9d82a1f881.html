Implicitly set the normalization constant for the functional.

The normalization constant is selected so that if an input IceModelVec2V has component vectors all of length scale, then the funtional value will be 1. I.e.

\[ c_N^{-1} = \sum_{i} w_i {\tt scale}^2. \]
