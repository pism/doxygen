Detailed Description

Ocean model "modifier" that helps with initialization.

This modifier saves all fields a ocean model provides as a part of the model state and re-loads them during initialization so that they are available before the first time step in a re-started run.

It is

  • not visible to the user,
  • is added automatically, and
  • does not have a corresponding "keyword" in ocean::Factory.
