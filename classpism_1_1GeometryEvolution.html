Detailed Description

NB! Write this in a way that does not use ghosts of input fields (copy to temp. storage and communicate).

The promise:

H_change + Href_change = dt * (SMB_rate + BMB_rate - flux_divergence) + conservation_error

Href == 0 if H > 0
