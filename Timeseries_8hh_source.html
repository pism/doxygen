34 //! \brief A general class for reading and accessing time-series.
35 /*!
36  \section timeseries_overview Scalar time-series
37 
38  This class provides random access to time-series values. It is used to
39  implement forcing with scalar time-dependent parameters (such as
40  paleo-climate forcing).
41 
42  \subsection timeseries_example An example
43 
44  The following snippet from PAForcing::init() illustrates creating a Timeseries
45  object and reading data from a file.
46 
47  \code
48  Timeseries delta_T(grid, "delta_T", "time");
49  delta_T.variable().set_string("units", "Kelvin");
50 
51  delta_T->read(dT_file);
52  \endcode
53 
54  Call
55  \code
56  double offset = delta_T(time);
57  \endcode
58  to get the value corresponding to the time "time", in this case in years. The
59  value returned will be computed using linear interpolation.
60 
61  It is also possible to get an n-th value from a time-series: just use square brackets:
62  \code
63  double offset = delta_T[10];
64  \endcode
65 */
